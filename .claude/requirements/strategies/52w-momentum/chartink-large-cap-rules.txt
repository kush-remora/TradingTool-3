and Daily count( 30, 1 where abs ( daily close - daily open ) / daily open * 100 <= 3 ) >= 24
and Abs ( Daily count( 30, 1 where daily close > daily open ) - Daily count( 30, 1 where daily close < daily open ) ) <= 4
and Daily count( 30, 1 where ( daily high - daily low ) / daily open * 100 >= 5 ) <= 3
and Daily Ema ( ( Daily High - Daily Low ) / Daily Open * 100 , 30 ) <= 3.5
and Daily count( 30, 1 where daily volume < daily sma ( daily volume , 30 ) ) >= 21
and Daily count( 30, 1 where daily volume < 1 day ago volume ) >= 15
and Daily countstreak( 30, 1 where daily volume <= 1 day ago volume ) >= 3
